Сайт был создан для учеников девятых классов, которые испытывают трудности в подготовке к ОГЭ по обществознанию. Нажав кнопку "Кодификатор ⇪", актуальный на данный момент, кодификатор загрузится на ваше устройство в виде Pdf-файла. Следующие кнопки переносят на страницы с самыми базовыми и необходимыми терминами, которые находятся в разных разделах. В разделе "Тесты" находятся тесты для закрепления терминов. Для возвращения на главную страницу можно нажать на логотип сайта.
